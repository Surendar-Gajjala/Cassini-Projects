CREATE MANUFACTURER

Steps to create Manufacturer:

 

1.      Go to Procurement tab in the navigation bar

 

 

 

     2.     Click on Manufacturers link

 

 

3.      Manufacturers page will display

4.      Click on New Manufacturer button

5.      New Manufacturer page opens

6.      Fill all the mandatory fields

7.      Click on Create button

 

 

Manufacturer will be created successfully.
