FAILURE REPORT

 

Failure Report will be done on Issued parts only.

 

To this we need to follow below steps:

 

1.      Go to BOM page

2.      Click on BOM Info button

3.      Select any Missile

4.      Click on Issued parts UPN

 

 

 

 

5.      Click on Create Failure Report button

 

 

 

Failure Report page will display

 

1.      Fill in all the details

2.      Click on Create button

 

 

 

Failure Report will be created.

 

 

 

1.      We can update the data of all the fields except date

2.      Data will be updated by clicking on Update button

3.      BDL will give the approval with BDL Login

 

 

SSQAG will give the approval with SSQAG Login

 

 

CAS Manager will give the approval with CAS Manager login.

 

 

Similarly Store Manager also approve the list.

 

We can print the Failure Report by clicking on Print button at the top.
