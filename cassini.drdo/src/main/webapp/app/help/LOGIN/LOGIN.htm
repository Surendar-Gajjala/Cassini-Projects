LOGIN:

Let’s start with Login to CAS.MS web application.

 

Steps to Login to CAS.MS:

 

• Open the Browser - Chrome / Firefox

• Type in the CAS.MS URL in the browser

• User will see the Login Screen

 

 

1. Fill in the User ID

2. Fill in the Password

3. Click the Login Button

 

User will be successfully logged into the CAS.MS application

 

Note: If you are unable to login, please contact the Cassini Admin
