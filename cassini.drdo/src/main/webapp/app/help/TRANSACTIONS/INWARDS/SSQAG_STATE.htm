SSQAG STATE

SSQAG will open the Inward for quality check:

 

SSQAG will have 4 options on each and every part

 

è He can Accept the Part

è Partial Accept the Part

è Reject the Part

è Review the Part

 

Above all steps will be done by below steps.

 

1.      Go to Inwards tab in the navigation bar

 

 

 

2.  Inwards page will be displayed

3.    Click on Inward Number hyperlink

 

 

 

Inward details page will be opened with SSQAG state on the top progress bar.

 

 

4.      Click on Accept button to accept the part

5.      Click on Partial Accept button to partially accept the part

6.      Click on Reject button to Reject the part

7.      Click on Review button to Review the part

 

 

Except for Accept state remaining 3 states we need to mention the reason for Partial Accept/Reject/Review part.

 

Then the Inward will be submit to Inventory Manager/Store Manager.

 

Inventory Manager/Store Manager Login :

 

Inventory/Store Manager will open the Inward and will do the following steps:

 

1.      For SSQAG accepted Part-> he will allocate Storage location by clicking on “Allocate Storage” button

 

 

 

 

Upon clicking on Allocate Storage button-> Inward storage location will be allotted for the accepted part

 

 

 

2.      Once the Part is allotted, Store manager will verify the Item by clicking on Verify Item Button

 

 

3.      Verify Item page will be opened

4.      Scan the UPN/Storage on the Part

5.      Click on Verify button

 

 

 

Part will be verified successfully.

 

 

 

 

Partial Accept Part:

 

For SSQAG Partial accept part -> Store manager can assign the Storage location same as the Accept Part case as mentioned above.

 

Or else, he will edit the part details and update the comments and will sent to SSQAG again for Review.

 

To do this we need to follow below steps:

 

1.      Click on Edit button on Partial Accept part

 

 

 

 

2.      Update any missing details on the part

3.      Click on Save

 

 

 

 

4.    Add Comments page will open up on clicking on Save button

5.    Enter the Comments

6.      Click on Add button

 

 

 

Part will be updated with latest details and also Re-submitted to SSQAG again

 

 

 

Reject state Part:

 

For SSQAG Rejected part, “Returned” storage location will be allotted automatically.

 

 

 

SSQAG will again Review the “Partial & Review” state parts which were updated by Store Manager.

 

We will do this by following below steps:

 

1. Go to the Inwards tab in the navigation bar

 

 

2.   Inwards page will be open

3.   Click on Inward Number hyperlink

 

 

 

4.      SSQAG will review and Accept the Parts (which he Partial Accept & Reviewed) by clicking on Accept button

 

 

 

Parts will be Accepted.

 

 

 

 

Then Store manager will get notified.

 

Store Manager will again allocate the storage to the accepted parts as mentioned above in the Accept Parts state.

 

With this Inward state will be Finished.  So our Inward will looks as below:

 

 

In the above picture 3 Parts were saved in Inward location and 2 Parts were saved in return location as per the above process.
