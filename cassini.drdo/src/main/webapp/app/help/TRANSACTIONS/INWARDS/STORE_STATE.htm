Inventory Manager/Store Manager will login to the CAS MS application and Inward the material by following below steps:

 

Steps to create Inward:

 

1.      Go to Transactions ->Inwards tab in the Navigation bar

 

 

2.      Inwards page will be displayed

3.      Click on New Inward button

4.      New Inward page opens

5.      Search the Gatepass

6.      Click on Add Gatepass button

 

 

7.      Fill in all the mandatory fields

8.      Click on Create button

 

 

New Inward will be created successfully.
