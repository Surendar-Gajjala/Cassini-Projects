GATEPASSES

Security State:

 

Security will login to the CAS MS application and create Gate pass.

 

There are 2 ways in creating the Gatepass:

 

->We have an option to create from Home page

->We have another option to create from Transactions-> Gatepass page

 

Steps to create New Gatepass:

 

1.      Go to Transactions -> Gatepasses tab in the navigation bar

 

 

2.      Gatepasses page will be opened

3.      Click on New Gatepass button

4.      New Gatepass page will be opened

5.      Received By field will be autoloaded by logged in person name

6.      Upload the Gatepass file(supports only PDF file)

7.      Gatepass Number will be auto filled with Gatepass file name

8.      Select Gatepass Date

9.      Click on Create button

 

 

Gatepass will be created successfully.
