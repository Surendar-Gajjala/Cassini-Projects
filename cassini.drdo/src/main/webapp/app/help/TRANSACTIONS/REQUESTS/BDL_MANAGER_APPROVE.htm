BDL MANAGER APPROVE

Once the Request created notification will be sent to BDL Manager. BDL Manager can Approve/Reject request.

 

Steps to perform BDL Manager Approve/Reject:

 

1.      Go to Requests page

2.      Click on Request Number

 

 

3.  Request details page will be opened

4.  Click on Reject button to Reject the Request (If the Request got Reject, then it will be last step for that request)

5.  Click on Approve button to Approve the Request

6.  We can also accept the Request by clicking on Accept button

 

 

Request will be approved once we click on Approve button and confirm Approve dialog.
