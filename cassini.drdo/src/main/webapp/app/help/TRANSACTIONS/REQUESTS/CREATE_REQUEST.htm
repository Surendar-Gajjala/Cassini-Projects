CREATE REQUEST

BDL Employee will Request for the material after that BDL Manager & CAS Manager will approve/reject the Request based on the requirement.

 

Steps to create Request:

 

1.      Go to Requests tab in the navigation bar

 

 

 

2.      Requests page will be displayed

3.      Click on New Request button

 

 

4.      New Request page will be opened

5.      Select the BOM

6.      Select the Instance (Missile)

7.      Request By field auto filled by Logged in user name

8.      Enter Purpose (Optional)

9.      Click on Close (X) button

 

 

 

9.      Search for the Part to be Request

10.      Click on Add Item (+) button

 

 

12.   Enter the qty. to Request

13.   Click on Create

 

 

Request will be created successfully.
