QTY. ALLOCATE ON MISSILE

Steps to Allocate qty. to Missiles:

 

1.      Go to Planning tab in the navigation bar

 

 

2. Planning page will be opened

3. Click on Planning Info button

4. BOM tree will be displayed

5. Select the System

 

 

6. Select the Missiles to be planned from the Missiles drop down

7. Select the Sections to be planned from the Sections drop down

 

 

8.      Expand the Section to its part level

 

 

 

9.      Click on Allocate button (+) on any Missile

 

 

10.      Allocation will be happened on selected Missile
