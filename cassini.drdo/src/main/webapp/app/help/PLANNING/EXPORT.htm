EXPORT

We can export the Planning module page to excel.

 

Steps to do Export:

1.      Go to Planning page

2.      Click on Planning Info button

3.      BOM tree will be displayed

 

 

4.      Select any system

5.      Select Missiles from the Missile drop down

 

 

6.      Select the Section to export from Section drop down

7.      Click on Export button

 

 

Planning page will be exported.
