GROUPS

Create Group:

 

1.      Go to Admin page

2.      RMB Groups -> Select New Group

 

 

 

3.   New Group creation page will open

4.   Enter the Group Name

5.   Enter the Description (Optional)

6.   Select Is Active

7.   Click on Create

 

 

New Group will be created successfully.

 

Update Group:

 

We can update the Group details, if we want to change further:

 

1.      Select any Group

2.      Click on Edit Group button

 

 

3.   Update the details

4.   Click on Update Group button

 

 

 

Group will be updated successfully.

 

Add Users to Group:

 

1.      Select any Group

2.      Click on Add Users button

3.      Select Users page will open

4.      Select the User

5.      Click on Add button

 

 

User will be added to Group successfully.

 

Delete Group:

 

Steps to delete group:

 

1.      Select any Group

2.      RMB Delete Group

 

 

3.   Delete Group confirmation dialog will open

4.   Click on OK

 

 

Group will be deleted successfully(if there are no members in the Group).

If there are members assigned to the group then we need to remove the user from the group and should then Delete the Group.

 

To Delete User from Group:

 

1.      Select the Group

2.      Click on Delete button in the Users section

 

 

3.      Click on OK in the confirmation dialog
