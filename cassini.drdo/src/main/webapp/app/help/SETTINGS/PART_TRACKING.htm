PART TRACKING

Part Tracking steps will be created at Settings -> Part Tracking Section.

 

Using these steps only BDL Manager/SSQAG/CAS Manager will proceed further with Part Tracking verifications.

 

Steps to create Part Tracking Steps:

 

1.      Go to Settings page

2.      Select Part Tracking option from Settings

3.      Click on New Part Tracking (+) button

4.      New Part Tracking window will display

5.      Enter the Process Name

6.      Click on Save

 

 

Part Tracking Process will be created.

 

Under this we need to create Part Tracking Phase.

 

Steps to create Part Tracking Phase:

 

1.      Click on Add Part Tracking Phase (+) button

2.      Add Part Tracking Phase window will be open

3.      Enter Status Name

4.      Enter Description

5.      Enter Percentage

6.      Select the Options (Attachment/Scan/BDL Manager/SSQAG/CAS verification checks) as per the requirement

7.      Click on Save

 

 

Part Tracking Phase will be created successfully.

 

Similarly, we can create more number of Phases.
