SYSTEM ATTRIBUTES

We can add any no. of system attributes to any type of object by following below steps:

Steps to add system attributes:

1.      Go to Settings page

2.      Select System Attributes option from Settings

3.      Go to Select Object Type drop down and select any one of Object Type

 

4.      Click on New Attribute (+) button

5.      New Attribute window opens

6.      Enter Name

7.      Enter description

8.      Select Data Type

9.      Select Required checkbox (If System Attribute is mandatory for the Object Type)

10.       Click on Save button

System attribute will be saved successfully.

 

Similarly, we can save more System attributes on the Object Type.

We can update the System Attribute by clicking on Edit button.

 

We can delete the System Attribute by clicking on Delete (X) button.
