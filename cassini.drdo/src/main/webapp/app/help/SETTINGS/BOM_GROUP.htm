BOM GROUP

BOM Group help us to create Section, Subsystem & Unit types for BOM creation.

 

Steps to create BOM Group Type:

 

1.      Go to Settings page

2.      Select BOM Group option from the Settings

3.      Click on New Type (+) button

4.      Select the Type (Section/Subsystem/Unit)

5.      Enter Name

6.      Enter Code (For Section, Subsystem-> code should be single digit and for Unit -> code should be two digits)

7.      Click on Save

 

 

BOM Group type will be saved successfully.

 

We can update the Type by clicking Edit button.

 

 

We can delete the Type by clicking Delete button.
