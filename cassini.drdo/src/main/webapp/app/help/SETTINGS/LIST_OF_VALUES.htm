LIST OF VALUES

List of Values:

Steps to create New List of Values:

1.      Go to Settings page

2.      Select List of Values option from Settings

3.      Click on New Lov (+) button

4.      Update the Lov name

5.      Click on OK button

 

 

LOV will be saved.

We can add a value to the LOV as we wish.

Steps to add a value to LOV:

1.      Click on Add New Value (+) button

2.      Enter the value name

3.      Click on OK button

 

Similarly, we can add more values to LOV.

So the LOV becomes as below.

 

 

 

We can update the value as well, by clicking on Edit button on the Value.

 

We can delete the Value by clicking on Delete Value button.

 

Similarly, we can Update and Delete LOV as well.

To update LOV, double click on LOV name and update.

To delete LOV, Click on Delete this LOV (X) button.
