AUTO NUMBERS

Settings module help us to set the values depending on user choice.

In this we are able to set various Auto Numbers, Life Cycles, List of Values, System Attributes.

 

Auto Numbers:

Steps to create New Auto Number:

 

1.      Go to Settings tab in the navigation bar

 

 

 

Settings page will be displayed.

 

2.      Select Auto Numbers section

3.      Click on New Auto Number (+ button)

4.      Fill in all the details

5.      Click on OK button

 

 

 

New Auto Number format will be created successfully.

We can perform Edit & Delete operations on created New Auto Number format.

 

Edit Auto Number:

 

1.      Select the Auto Number which we want to update.

2.      Click on Edit button

 

 

 

3.      Edited fields will be displayed, update the data

4.      Click on OK button

 

Auto Number will be updated.

Delete Auto Number:

 

1.      Select the Auto Number which we want to delete.

2.      Click on Delete button

 

 

3.      Delete confirmation msg will display

4.      Click on Yes
