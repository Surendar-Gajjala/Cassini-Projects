TARB DOCUMENT AT MISSILE LEVEL

Once each section is completed Part tracking, the MBOM is updated with all relevant attachments, part numbers and so on.

A TARB document template will be created by Admin.

And Users can generate the TARB document at MBOM or Section level.

 

Steps to generate TARB document at Missile level:

 

1.      Go to BOM tab in the navigation bar

 

 

2.      BOM page will be displayed

3.      Go to BOM tree section

4.      Expand the System and select any missile

 

 

5.      Select Missile and click on Generate TARB document button.
