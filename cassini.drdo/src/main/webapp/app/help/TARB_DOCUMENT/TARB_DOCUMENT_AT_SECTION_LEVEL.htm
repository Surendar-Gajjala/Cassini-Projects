TARB DOCUMENT AT SECTION LEVEL

Steps to generate TARB document at Section level:

1.      Go to BOM tab in the navigation bar

 

 

2.      BOM page will be displayed

3.      Go to BOM tree section

4.      Expand the system and select any missile

 

 

5.      Select the Section and click on Generate Section TARB document button
