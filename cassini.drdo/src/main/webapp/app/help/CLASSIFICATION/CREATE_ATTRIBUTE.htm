CREATE ATTRIBUTE

Steps to create New Attribute:

 

1. Select the Part Type

2. Click on New Attribute button

3. Enter the Name

4. Enter the Description

5. Select the Data Type

6. Select the “Is Required” check box (if the Attribute is mandatory for Part)

7. Click on Save button

 

 

Edit Attribute:

 

Steps to Edit Attributes:

 

1)     Click the EDIT button in the attribute’s actions field

 

 

 

 

2) Attributes fields will be editable, make any changes

3) Click OK to save changes

 

 

Delete Attribute:

 

Steps to Delete Attributes:

 

1)     Click the Delete button in the attributes Actions field

 

 

 

If the Attribute is used somewhere else, we cannot delete that attribute
