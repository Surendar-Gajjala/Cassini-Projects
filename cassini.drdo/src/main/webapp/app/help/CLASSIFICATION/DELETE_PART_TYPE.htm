Part Type Delete:

 

We can delete the “Part Type” if we need to be deleted. For this we have the below steps to be followed.

 

1. RMB Part Type

2. Select Delete Type

 

 

3. Click on “OK” button in the confirmation dialog

 

 

 

Part Type will be deleted successfully.

If Part Type is used somewhere else in the app, we cannot delete that Part Type.
