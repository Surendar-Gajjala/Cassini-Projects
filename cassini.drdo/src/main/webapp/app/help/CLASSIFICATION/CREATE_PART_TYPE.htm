CREATE PART TYPE

In this we are going to organize different levels of Item/Part Types using the following features:

 

Classification module has following steps to “Create or Delete Part Types”:

 

Create Part Type:

 

1.     Select the Classification tab in the Tool bar

 

 

 

2. Displays the Classification Tree on Left side

3. Select the Part Type to create its related Part Type

4. RMB Select Add Type

 

 

5. Enter the Part Type name and click on Enter button

 

 

 

6. Select the Part Type

7. Enter the Description

8. Enter the Code

 

 

9. Select the Auto Number Source from the drop down

10. Select the Revision Sequence

11. Select Units

12. Click on Save

 

 

For Lot Types also we need to follow the same process with the below additional steps:

 

1.     Select the Lot Type

2.     Enter the Description

3.     Enter the Code

4.     Select the Auto Number Source from the drop down

5.     Select the Revision Sequence

6.     Select Units

7.     Select Has Lots

8.     Select Has Specs

9.     Click on Add value

10.  Enter the value

11.  Click on Save

 

 

 

Similarly, we can create more Spec values for Lots by clicking on Add Value button.

 

We have one more option available for each Part Type:

 

1)     Has Expiration: - For Liquid Type Item Types there will be an Expiry date, so we will use this option if we want to add Expiry date for any Item Type.
