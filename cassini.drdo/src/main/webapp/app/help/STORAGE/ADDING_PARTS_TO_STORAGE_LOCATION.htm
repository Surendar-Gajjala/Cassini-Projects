ADDING PARTS TO STORAGE LOCATION

To add Parts to storage location, we need to follow below steps:

 

1.      Select any storage location

2.      Select “Is Leaf Node” option

3.      Enter “Total Capacity”

4.      Click on “Select Parts” drop down

 

 

5.   Select Parts page will display

6.   Select BOM tree structure

7.   Click on Add (+) button to add Items to Storage location

 

 

 

Close the Select Parts screen.

 

8.  Selected Parts will be added to Storage location

9.  Click on Save button

 

 

 

 

Storage location will be saved with selected Parts.

 

Similarly, we can create Storage locations for Return & Inward Storage locations.

 

For Return storage type, we need to select Return check.

 

For Inward storage type, no need to select any storage type.
