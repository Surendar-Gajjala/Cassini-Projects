CREATE STORAGE

This part will help us to create different storage levels to store the inventory items.

This will provide us the following features:

·        Create Storage Location

·        View list of all the Parts/Lots Stored in that Storage location

 

Steps to create Storage hierarchy:

 

1.      Select the Storage tab in the navigation bar

 

 

2.      Storage tree will be displayed

3.      Select the Storage tree, RMB select New Warehouse

 

 

4.      Enter the Name

5.      Enter the Description (Optional)

6.      Select BOM from the drop down

7.      Select the Storage Type (On Hold/Return Storage)

8.      Click on Save button

 

 

 

Warehouse storage type will be saved successfully.

 

Steps to create Stock Room:

 

1.      Select Storage tree or any Warehouse

2.      RMB select New Stock room

 

 

3.   Enter the Name

4.   Enter the Description (Optional)

5.   Select BOM from the drop down

6.   Select the Storage Type (On Hold/Return Storage)

7.   Click on Save button

 

 

Stock Room storage type will be saved successfully.

 

Steps to create Area storage type:

 

1.      Select Stock Room

2.      RMB select New Area

 

 

3.  Enter Name

4.  Enter Description(optional)

5.  BOM & Storage types will be auto loaded from parent storage type

6.  Click on Save

 

 

 

Area storage type will be saved successfully.

 

Steps to create Sub Area:

 

1.      Select Area

2.      RMB select New Sub Area

 

 

3.  Enter Name

4.  Enter Description (optional)

5.  BOM & Storage types will be auto loaded from parent storage type

6.  Click on Save

 

 

Sub Area storage type will be created successfully.

 

Steps to create Rack:

 

1.      Select Area/Sub Area

2.      RMB select New Rack

 

 

3.  Enter Name

4.  Enter Description (optional)

5.  BOM & Storage types will be auto loaded from parent storage type

6.  Click on Save

 

 

Rack storage type saved successfully.

 

Steps to create New Shelf:

 

1.      Select any Rack

2.      RMB select New Shelf

 

 

3.   Enter Name

4.   Enter Description (optional)

5.   BOM & Storage types will be auto loaded from parent storage type

6.   Click on Save

 

 

Shelf storage type saved successfully.

 

Steps to create Bin storage type:

 

1.      Select any Shelf

2.      RMB select New Bin

 

 

3.    Enter Name

4.    Enter Description (optional)

5.    BOM & Storage types will be auto loaded from parent storage type

6.    Click on Save

 

 

Bin storage type saved successfully.
