OVERVIEW

Inventory Management is managed on the BOM data. So CAS.MS provides ways to create and manage BOM in CAS.MS

CAS.MS provides the following functions

 

·       Load Existing BOM and view hierarchical BOM Structure

·       Create New BOM - With Parts, Assemblies and Sub-assemblies

·       Manage BOM - Add, Modify, Delete

·       Create MBOM from EBOM

 

Definitions:

EBOM: Engineering Bill of Material, usually this BOM is created by R&D / Engineering team during design phase.

 

MBOM: Manufacturing Bill of Material, this BOM is created by manufacturing planning team for the production phase.

 

From one EBOM, users can create multiple MBOMs.

 

Example: We get an EBOM from R&D team and we get order to build 10 missiles, so we have to create 10 MBOMs for 10 Missiles using the EBOM.

 

Overview

BOM Page displays the BOM details and also helps users to manage BOM. Some of the core functions in the BOM Management are

·        Create BOM

·        Create and Add Parts to BOM

·        Display BOM

·        Edit BOM Parts

·        Delete BOM Parts
