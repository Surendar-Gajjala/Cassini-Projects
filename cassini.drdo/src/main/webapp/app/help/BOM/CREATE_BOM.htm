CREATE BOM

We can create BOM in following ways:

·        Import BOM

·        Create BOM

 

Import BOM

Cassini systems supports you to migrate your BOM data from legacy systems into CAS.MS. Please contact us with your BOM data so we can help you migrate your BOM data into CAS.MS.

 

Create BOM

Steps to create BOM

1.      Select the BOM in the navigation bar

 

2.      BOM Tree will be displayed with all the Systems

 

Add Section type to BOM:

 

3.  Select the System

 

 

 

4.      Click on Add Section button

 

 

 

     5. Add Section screen will be displayed

     6. Click on Add Item button

 

 

 

 

7.    BOM Type (Section) will be added.

8.    Click on Close button

 

 

 

Add Subsystem type to BOM:

 

1.      Click on Add Item button on Section

2.      Select Add Subsystem

 

 

 

3.    Add Subsystem page will be displayed

4.    Click on Add Item button

 

 

5.      Subsystem will be added to Section BOM

6.   Click on Close button

 

 

 

  Add Unit Type to BOM:

 

1.      Click on Add Item button on Section/Subsystem

2.      Click on Add Unit

 

 

3.  Add Unit screen will be displayed

4.  Click on Add Item button

 

 

 

5.  Unit type will be added to BOM

6.  Click on Close button

 

 

 

Add Part type to BOM:

 

We have 2 ways in adding the Part to the BOM

 

1st way of doing, please follow the below steps:

 

1.      Click on Add Part button on Subsystem/Unit

2.      Add Part screen will be displayed

3.      Search for the Part to be added (with the Nomenclature/Code/Type fields)

4.      Click on Add Item button

 

 

5.    Part will be added to selected Subsystem/Unit

6.    Click on Close button

 

 

 

7.  Enter the Part qty.

8.  Click on save

 

 

Part will be added to BOM.

 

 

 

 

2nd way of adding Part type to BOM:

 

1.      Click on Add Part button on Subsystem/Unit ->Add Part screen will be displayed

2.      Click on New Item button

 

 

 

3.   Fill in all the mandatory values

4.   Click on Add button

 

 

5.   Part will be added to BOM

6.   Click on Close(X)

 

 

Similarly, we can add/create any number of parts/Unit/Subsystem/Section to BOM.
