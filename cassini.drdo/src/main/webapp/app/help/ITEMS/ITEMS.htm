ITEMS

CAS. MS maintains all the Item/Part related data in the Item module.

 

Steps to create Item:

 

1.     Go to Items module in the tool bar

 

 

2.     Items page will be displayed

 

 

 

3.  Click on New Item button

4.   New Item screen will be opened

5.   Select Type

6.   Nomenclature & Item code will be filled automatically

7.   Enter the Description

8.   Units will be fill automatically

9.   Enter the Material

10. Enter the Drawing Number

11. Click on Create button

 

 

New Item will be created. Similarly, we can create any number of items.

So the page looks like as below:

 

 

On any item we have 4 options in item details view.

 

i)                 Basic Info :- which will contain all the item information, we can update Nomenclature, Description, Drawing Number & Material details if we want to update further. To update the value, click on the field to be updated. Updated the data and click on save.

 

 

ii)  Attributes view : Attributes which are added while creating Item Type will be displayed here.  We can update the Attribute value if we want to change further. To update the value, click on the field to be updated. Updated the data and click on save.

 

 

iii)  Files: We can add files for Items. To do this we need to follow below steps:

1)     Go to files tab

2)     Click on the hyperlink “Click here to add files”

3)     Select the file from local PC

 

 

File will be uploaded/added successfully.

 

 

 

We can also view the File history by clicking on File history button in the actions field.

We can also delete the file by clicking on Delete button in the actions field.
