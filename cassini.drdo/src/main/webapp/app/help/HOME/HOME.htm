HOME

Home screen gives access to all the available functionality of the CAS.MS application based on your access privileges.

 

1. Home

2. Tool bar - Tools will be displayed as per user access privileges

3. Login User Details

4. Widget Area 

5. Widgets - Gate Passes, Inwards, Inward Items, Requests, Issues, Returns, Failures & Dispatches

 

 

GatePasses Widget: ability to create Gate Pass for the material received and also ability to view all the created GatePasses in this widget

 

 

Inwards Widget: ability to create New Inwards also displays all the material Inwards which we have procured from Suppliers and other manufacturers

 

Inward Items Widget: displays all the Inward Parts to perform the proceeding operations for SSQAG/STORE

 

Requests Widget: ability to create New Request, also displays all the created Requests which the users Request Material for the project.

 

 

 

Issues Widget: ability to create New Issue, also displays all the material Issue details based on the Requests received

 

 

Returns Widget : All the Returned parts will be able to view here.

 

 

Failures Widget: The parts which were completed Failed process will be able to view in this widget.

 

 

Failure Process Widget: The parts which are in Failed process will be able to view in this widget.

 

 

 

Dispatches Widget: ability to create New Dispatches for Returned, Failed & Fabrication Parts, also displays all the Returned, Failed & Fabrication parts dispatches here.

 

 

 

 

Expiry Items Widget : In the Inward process we can enter Expiry date for the parts(optional) as we wish to enter. Once the given date has expired on the part then those parts will be able to view in this widget.

 

 

The above all widgets will be available to view or create based on the permission levels.
