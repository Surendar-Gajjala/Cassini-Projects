OVERVIEW

Executive Summary is used for "Missiles & Sections" status/tracking.

 

In this module we are able to see the Sections below status:

 

-> How many Sections are completed

-> How many Sections are in Progress

-> How many Sections not yet started

 

In this module we are able to see the Missiles below status:

 

-> How many Missiles are completed

-> How many Missiles are in Progress

-> How many Missiles not yet started
