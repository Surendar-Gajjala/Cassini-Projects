PART TRACKING

Part Tracking will be done from the BOM missile.

 

We will create Part tracking process steps in the “Settings-> Part Tracking section”

 

Part tracking can be done on Section or Missile levels. Both have the same functionality.

 

This will be done by “BDL Manager/SSQAG/CAS Manager/Store Manager”.

 

We need to follow below steps to perform Part Tracking.

 

1.      Go to BOM page

2.      Click on BOM Info button

3.      Select any Missile

 

 

 

 

4. Missile level BOM will be displayed

5. Click on Assign Part Tracking button

 

 

 

6.   Assign Part Tracking page will be opened

7.   Select the Part Tracking process

8.   Click on Add button

 

 

9.      Click on View Part tracking button

 

 

 

10.  Part Tracking process page will be displayed

11.  BDL will enter the comments and accept the states

 

 

 

Once BDL accepted the state, below is the screen we will get.

 

 

 

 

Then SSQAG will login and verify the status.

 

SSQAG can Accept / On Hold the state, If the SSQAG keep the state as On Hold then it will not move forward further.

 

If the SSQAG accepts then it will go for CAS verification.

 

 

Once SSQAG accept the state below is the screen we will get

 

 

Then it will go for CAS manager verification.

 

CAS Manager can Accept/On -Hold the state. If the CAS Manager keep the state as On-Hold then the process will not move further.

 

Once the CAS Manager verified below is the screen we get.

 

 

 

This process will continue further with all the steps.

 

SSQAG/CAS Managers who put the state in On-Hold will further release and continue the process.

 

1.      We can update the comments by clicking on Edit button

2.      We can upload the attachments as well by clicking on Choose file button.

 

 

 

 

3.      We can print the Part Tracking process by clicking on Print pdf icon

 

 

Print page will be displayed

 

 

4.      Percentage of Part Tracking Completed will be updated in the Percentage field.
