MY PROFILE

 

Steps to view My Profile

 

1.    Login to the CAS MS

 

2.    Select the Logged in User Name (To the Top Right Corner)

 

3.    Select the My Profile

 

4.      Displays user details

 

5.    Select the Change Password

 

6.    Change Password dialog opens

 

7.    Fill in details

 

8.    Click on Submit button
