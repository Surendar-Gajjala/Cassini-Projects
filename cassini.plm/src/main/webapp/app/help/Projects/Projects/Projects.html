What is Projects?

            A project is an endeavour with lot of activities, tasks, deliverables and many more information that required to accomplish project goals. In Cassini.PLM user can create projects with planning, activities & tasks, project files, deliverables and required items etc.

 

What is the use and why do we need to create Project?

            Creating a project is useful to plan activities, tasks, milestones and deliverables, user can manage all project activities information and can manage multiple projects with the help of this functionality.

 

How to create a ‘Project’?

1.Click on "Navigation" icon

2.Click on "Projects"

3.Select"Projects", Projects window will open

 

4.Click on "New Project" button, Popup will appear

5.Enter all the Mandatory fields (marked as ‘*’)

6.Click on Create, Success message will appear on user screen once it created successfully

 

 

 

How to Search /Edit / Delete a Project?

 

Search Project

1.Enter Project name into Search texbox

2.Click on search,requested project will display from the projects list

3.Click on Search Clear(X),to cleat the textbox text

Edit Project

1.Click Edit button to the related project, Popup will appear.

 

2.Modify required filed would you like Change.

3.Click on "Update", Will get success message

 

Delete Project

1.Click on delete button related to the project

2.Will get confirmation dialogue box

3.Click on "OK" will get success message
