What is Activity?

Activity is next level of breakdown of project works, user can create activities within a WBS and each activity will be associated with multiple tasks and contains different files and deliverables.

 

Why do we need to create Activities?

Activity creation is a mandatory action, without creating activities user cannot create tasks.

 

How to create Activities?

 

1.    Select Add Activity button

2.  N ew Activity panel opens

3.      Fill in all the details

4.      Click on Create Activity button

Activity will be created successfully.

 

How to Edit Activity?

 

1.    Select Edit Acivity against an activity

2.    Edit Activity page opens

3.    Update the details

4.    Click on Update Activity button

Activity will be updated successfully.

 

How to Delete Activity?

 

1.    Select Delete Acivity button from the action against each activity

2. Confiramtion dialouge will display

3. Click on OK
