What is WBS (Plan)?

WBS is Work Breakdown Structure used to plan the project works/activities, WBS helps to breakdown project works into small workable tasks which can be easily tracked. Here user can plan all project works, activities and milestones which indicates which work need to be completed in which milestone. With the help of WBS user can easily plan project works.

 

Why do we need to create WBS or Plan?

Without creating WBS user cannot create activities and add tasks to each activity. It is mandatory to create WBS to create activities and tasks, then users can perform and update project tasks.

 

How to create WBS or Plan?

 

1. Got to plan tab in project details

2. Click on Add WBS

3. New WBS page will display

4. Enter the details

5. Click on Create WBS

WBS will be created successfully.

 

How to Edit WBS?

 

1.    Select 'Edit' WBS option from actions against each WBS/Activity line

2.    Update the details

3.    Click on Update WBS button

WBS will be updated successfully.

 

How to Delete WBS?

 

1. Select 'Delete' WBS option from actions against each WBS/Activity line

2. Confirmation dialogue will open

3. Click on OK
