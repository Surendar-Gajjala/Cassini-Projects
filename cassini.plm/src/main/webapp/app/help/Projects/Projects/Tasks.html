What is Task?

            Tasks are small executable pieces of the project work; multiple tasks can be created on an activity. These tasks will be assigned a team member and they can update the percentage of work completion, managers and other will know status of each tasks and completed portion of the work.

 

Why do we need to Create Tasks?

            Creating tasks and assigned it to team member will helps in identify the work status and as well as project status. It is easy to track and maintain multiple project works and all project information will be at user fingertips.

 

How to Create Tasks?

 

1.    Click on an Activity

2.      Click on Add Task button or "+" symbol

3.    New Activity Task panel opens

4.    Fill in all the details

5.    Click on Create Task button

 

How to Update a Task?

 

1.    Select Edit Task from Actions button

2.    Type number or use up & down arrows to increase or descrease task percentage

3.    Select Save Task from Actions button

 

Task will be updated successfully.

We can directly click on Finish button to update the complete Task in a single step.

 

How to add Files to a Task?

 

1.    Select Files tab

2.    Click on Add Files or Drag and Drop the files from local storage

3.    Select files from local storage

4.      Click on Open

Files will be uploaded successfully

How to Replace File?

How to Download files?

2.    Click on a file name and  Save file into local storage
