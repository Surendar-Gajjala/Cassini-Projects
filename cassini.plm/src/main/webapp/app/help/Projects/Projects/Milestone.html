What is Milestone?

            In projects, team should work on different activities/tasks to reach project goals or objectives, in this process milestone acts as laps, every milestone is like a lap and it helps management / stakeholders to know the status of the project.

Why do we need to create Milestone?

            It is not mandatory to create milestones in this application, but it helps users to know the project status.

How to Create Milestones?

 

1.  Select Add Milestone

2. New Milestone panel opens

3. Enter required details (Name, Description, Assigned To and Planned Finish Data)

4. Click on create milestone

 

How to Edit Milestones?

1. Select Edit Milestone from actions button

2. Edit Milestone panel opens

3. Edit required details

4. Click on Update Milestone

 

How to Delete Milestone?

 

1. Select Delete Milestone from actions button

 

2. Confirmation dialouge will diaplay, click on "OK"
