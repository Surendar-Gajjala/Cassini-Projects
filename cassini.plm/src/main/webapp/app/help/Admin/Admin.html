What is the functionality of ADMIN?

            Admin tab has users, groups, permissions creation functionality. We can create users and add them to groups and assign different permissions to access the application.

How to create Users?

 

1. Select the Admin from navigation bar

2.      Admin page will be displayed

3.        Select the Users and RMB ( Select New User)

6.      New user screen displays

7.      Fill in all Details

8.    Click Create

 

How to create Groups?

 

1.   Select Admin from navigation bar

 

2.      Select the Groups and RMB ( Select New Group)

 

3.    New Group screen open on right

4.    Fill in details

5.    Select the Select Users Button

6.    Select Add Users

7. Select one or many users

8. Click on Select, (Selected users add to group)
