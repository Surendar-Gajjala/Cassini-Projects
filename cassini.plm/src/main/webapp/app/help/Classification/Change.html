What is CHANGE type?

            Change type is also a pre-defined object in classification which can be used to classify the type of change it can be an engineering change or manufacturing change user can create change type and sub types.

 

Why do we need to CREATE change type?

            Creating change type will help user to classify different changes into dedicated types.

Example:User can create an Engineering change type and create call engineering change orders (ECO) under that type same for manufacturing and others.

 

How to CREATE/DELETE Change type and subtype?

CREATE Change type

 

1.       Select the Classification Node or any existing Change Type

2.   Click on Right mouse button then Add Type

 

3.       Edit and Fill in Change Type Name, Press enter

 

DELETE Change type

 

1.     Click on Change Type RMB (Click on Delete Type)

Note:   If the Change type has any objects already created it is not possible to delete type.

First user must manually delete all objects of the Change type and then delete the Change Type.
