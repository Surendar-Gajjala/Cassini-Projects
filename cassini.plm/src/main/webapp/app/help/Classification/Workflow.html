What is Workflow type?

              Workflow is used to define the multi process steps to complete a work, various workflows will be there for several operations. Some of the relevant workflows will be measured as group and we can call that group as a workflow type.

 

Why do we need to Create Workflow type?

             It is important to create workflow type, before creating a workflow user should define the workflow types. All workflows that are going to be created in the system must connect any one of the workflow types.

 

How to CREATE/VIEW/DELETE workflow type?

CREATE NEW WORKFLOW TYPE

 

1.       Right click on "Workflow" (Add Type)

2.       Edit and Fill in Workflow Type Name.

 

DELETE WORK FLOW TYPE

 

1.     Select the workflow Type, click Right mouse button,then Delete Type
