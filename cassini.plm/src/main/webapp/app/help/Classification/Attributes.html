What is Attributes?

            Attributes are nothing but additional properties of an object type, a user can create any number of attributes on each object type.

If a user wants additional property on an data type, he/she can create it using attributes feature, It provides various benefits like supporting different object types and user can opt for options like mandatory/optional, visible/hide, change control, list control and etc…

 

Why do we need to CREATE Attributes?

            Creating attributes on a classification object type is not a mandatory action, it is an extra feature that helps user to create additional properties on an object type.

Example:An electronic item called “Wire” is exited in the system. if a user wants to specify the colour of the wire sleeve, he/she can add an additional property on its object type and specify colour code or name. In this way user can add any number of properties.

 

How to CREATE/DELETE Attributes?

CREATE ATTRIBUTES on Item type

 

1. Click on Item Type

2. Click on "New Attribute".

3. Enter "Name" (Mandatory).

4. Enter "Description"(optional).

5. Select "Data Type" (Mandatory).

6. Select "Is Required" (optional).

7. Click on "Create".

8. Click on cancel to exit from the attributes.

 

 

DELETE ATTRIBUTES on Item type

 

1. Click on Item Type.

2. Click on DELETE icon of an attribute

 

3. Click on OK to confirm, (Attribute deleted successfully)

 

Note:If an attribute already in use cannot be deleted because that attribute is associated with some objects. There is only one way to delete used attribute, manually delete all objects those have attribute that user wants to delete and then delete the attribute.
