Classification

What is Classification?

            Classification is used to classify different objects like Items, Changes, Requirements, manufactures and manufacturer parts etc… User can create multiple object types and any number of attributes on each type.

 

Why do we need to CREATE Classification?

            User should create classification type before creating any new object (Item, Workflow, Change, Manufacturer, Manufacturer part etc.). For example, if a user wants to create a new item, he/she should create item type first in classification and then only can create items.

Creating classification will help users to classify objects under respective types

Example:All Mechanical items can be created under mechanical type and all Electronic items can be created under electronic type.

 

 

What is ITEM type?

            Item type is a pre-defined object in classification which can be used to classify all items/parts.

Why do we need to CREATE Item type?

            Creating Item type will help user to classify items into respective types. As mentioned in the previous example user can classify all mechanical items into mechanical item type and all electronic items into electronic item type.
