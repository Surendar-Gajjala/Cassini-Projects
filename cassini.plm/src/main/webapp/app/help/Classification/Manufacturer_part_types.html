What is MANUFACTURER PART type?

 

            This is also one of the pre-defined type in classification. Manufacturing part type is helpful in categorizing parts supplied by manufacturers based on its category.

Why do we need to CREATE manufacturer part type?

 

            Creating manufacturer part type will help users to specify which manufacturer will supply which types of items/parts/material.

Example: Manufacturer (ABC) supplies both Mechanical and Electrical, now user can categorize or create mechanical parts under mechanical part type and electrical parts under electrical part type of ABC organisation.

 

CREATE NEW MANUFACTURER PART TYPE

 

1. Right Click on "New Manufacturer Part", Click on "Add Type".

2. Enter Name (Press Enter, Manufacturer type created successfully)

 

DELETE MANUFACTURER PART

 

1.Right click on "Manufacturer Part Type" (Click on "Delete Type")

 

2.Confirmation dialogue will appear click on "OK",will get message.
