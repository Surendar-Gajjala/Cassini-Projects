Login

Let’s start with Login to Cassini.PLM web application.

 

•   Open the Browser (Chrome / Firefox recommended)

 

•   Type in the application URL in the browser (Example: - https://plm.cassiniqa.com)

 

•   User will be directed to Login Screen

 

 

How to Login to PLM application?

 

1.   Fill in the User ID

 

2.   Fill in the Password

 

3.   Click on Login Button

User will be successfully logged into the Cassini.PLM Web application

 

Note: If you are unable to login, please contact the Admin
