What is Lock/Copy/Delete items?

            Action button provides Lock, Copy and Delete item feature.

Lock: User can lock items with Lock item features, this feature will not allow other users to take any action on locked items.

Copy: User can copy items and create same item with a different number, this feature will save user’s valuable time in creating similar items.

Delete: If user wants to delete items this feature will help user to delete items from Items permanently. Deleted items cannot be recalled, so users should be cautious before deleting an item.

Why do we need to Lock/Copy/Delete items?

            Item Lock, Copy, Delete options are additional features on Item.

Lock: If an item needs secure access then permitted user can lock that item. After locking an item will be locked and users cannot perform actions on that item.

Copy: If user wants to create a similar item that is already existed in the system, user can use copy functionality and create same item with different item number. This feature will save user time from creating same item again.

Delete: This functionality used to delete items, if user wants to delete items he/she can use this Delete item feature to delete items.

How to Lock/Copy/Delete items?

Lock Item:

1. To lock Item click on lock button,will get item locked successfully message

2. To unlock click on "Unlock " button,it will show success message

Copy Item:

3. To copy the item click on Copy button,Popup will appear

4. Click on "Auto" Item number will generate, enter description

5. Click on "Create"

 

Delete Item:

6. To delete item click on "Delete" button

7. Confirmation dialogue will appear,Click on OK,success message will appear
