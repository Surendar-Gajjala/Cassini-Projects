What is the use of ‘Files’?

            One item will have many files, like design documents, specification documents and many other related files. All these files can be uploaded here, and users can access these files anytime and anywhere.

 

How to Add & view Files?

 

1.    Select the an Item from Items List

2.    Select the Files Tab in Item Details page

3.    Displays any files already added

4.    Click on add files link or drag & drop files into the table to add new files
