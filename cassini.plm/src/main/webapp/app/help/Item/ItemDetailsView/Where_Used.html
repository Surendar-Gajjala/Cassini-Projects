What is ‘Where Used’ tab is containing?

            An item may used in any other assembly or sub-assembly or at anywhere in the product application will capture that details and list parts or entire structure here in Where Used tab. With this information users can access to the other areas of this part is being used.

User doesn’t need to take any action here. If a part is used somewhere in the process automatically application will capture it and display details in Where Used tab.

 

How to view Where Used?

 

1.       Select the an Item from Items page

2.          Item details page displayed, Select the Where Used tab

3.      If Part is used in any where in the BOM that will be displayed
