What is the use of Related Items?

            Related items tab contains items from related item types, user can define these relationships in settings tab.

 

Why do we need to add related items?

            Adding related items are not a mandatory action. If user add related items, other team member can access this information to identify the other related parts in this project.

 

How to Add items to Related Items?

 

1. Click on Related Items tab

2. Click on Add Item button

3. Create Related Item panel opens

4. Select Relationship (Created Relationships in Settings will be displayed here)

5. Select Parts

6. Click on Add (Selected items will be added successfully)

 

 

How to Delete items from Related Items?

 

1. Go to related Items tab

2. Select Delete Related Item from Action button

3. Confirmation popup will appear, click on ‘YES’ (Item will be deleted successfully)
