Item details View

            Item details view have many tabs and each tab have lot of data related to that item. Item details view has some icons (Subscribe, Share, Refresh) at the right-hand side of the screen, each of this icon has its own functionality. Let’s have a look

 

How Subscribe icon works?

1. Select an item

2. Click on Subscribe icon to follow, to know what are the actions placed on this item (Item Subscribed successfully).

 

How Share icon works?

1.  Click on Share icon

2. Share item panel opens

3. Select Share type

4. Select User/Group

5. Select Permissions

6. Click on Share button (Item Shared Successfully).

 

How Refresh icon works?

1.  Click on Refresh icon. Page will be refreshed, and details will be updated.

 

What is the use of Revision History?

            Revision history button displays selected item revision history. Entire item revision will be captured and displayed here. User can select different revisions of item and its details in revision.

 

How to View and how Revision History will work?

 

1.  Click on Revision History tab

2. Revision History panel opens

3. All revisions data will be displayed

4. Click on Revision name to view that revision item details

 

What is the use of Save As?

            User can copy this item and create a new item with different item number.

 

How Save As works?

 

1.  Click on Save As button

2. Copy item panel opens

3. Click on Auto number

4. Enter Description

5. Click on Create tab (Copy of this item will new item number will be created successfully)

 

What is use of Crete Change?

            User can create ECO’s from item details view. An ECO can be easily created here and from the Changes.

 

How Create Change works?

 

1.  Click on Create Change

2. New ECO panel opens

3. Fill in details

4. Click on Create (ECO will be created successfully, and user will be directed to ECO page)
