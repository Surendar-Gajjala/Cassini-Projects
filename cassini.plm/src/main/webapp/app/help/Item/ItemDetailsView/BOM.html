What is BOM?

            BOM is 'Bill of Material' that required to manufacture a product or goods. User can select and specify entire BOM of a product/assembly/sub-assembly/item. Complete list of items will be displayed in BOM tab, user can access this at any point of time.

 

Why do we need to create BOM?

            BOM creation is important and structure should be created by the user, this BOM structure helps other users to identify list of items that they need to use in their manufacturing process.

 

How to create BOM? [Add Signle]

 

1.    Select BOM.

2.      Click on "+" symbol to add new Item to BOM

3.    Select Single to add Single parts

4.      In Single item selection, click on search symbol

5.      Items list will appear, select item form the list

6.      Click on Add

7.      Selected part will be added to the BOM tab

8.      Fill in the details Quantity, Ref Des, Notes

9.    Click on right button to add entry

 

Add Multiple Items to BOM

 

1.    Click on "+" symbol to add item

2.         Select Multiple

3.    Multiple Item Selection panel opens

4.        Select Items

5.        Click on Add

6.      Fill in the details Quantity, Ref Des, Notes.

7.    Click on right symbol to save items.
