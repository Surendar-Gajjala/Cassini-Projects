What is Attributes?

            Attributes are nothing but additional properties on item, a user can create any number of attributes on each item.

If a user wants additional property on an item, he/she can create it using attributes feature, provided in classification. It has various benefits like supporting different object types and user can opt for options like mandatory/optional, visible/hide, change control, list control and etc…

 

Why do we need to Create Attributes?

            Attributes are useful in adding additional property on item. User can create attributes in classification, this is an additional & optional feature that helps user in creating additional properties on item types.

 

How to CREATE Attributes?

 

Created attributes in classification - item type will be displayed here. User can set value on these attributes when creating new Item.

1.    Click on New Item

2.    New Item panel opens

3.    Fill in Attribute values

4.    Click on Create

 

 



How to View Attribute Value?

1.    Select created items

2.    Select Attributes

3.    Attribute values will be displayed

 

How to Edit Attribute Value?

1.    Click on edit Attribute

2.    Edit value

3.    Click on "right" symbol (Value will be saved successfully)
