What is the use and Why do we need to add “Manufacturer Parts”?

            This tab is useful to add any manufacturer parts that are required to complete the work. User can add manufacturer parts here and rest of the team will have access to it based on access permissions. Adding manufacturer will clear, what are the items required and who is manufacturer with the manufacturer part number details etc…

 

How to add items to Manufacturer Parts?

 

1.    Select Manufacturer Part tab

2.    Click on Select Manufacturer Parts

3.    Manufacturer Parts Selection panel opens

4.    Select parts and

5.    Click on Add button (Selected manufacturer parts added successfully)

 

How to delete Manufacturer Part?

1.    Select Delete Part from Actions button

2. Confirmation popup will appear, Click on ‘OK’ ( Item will be deleted successfully)
