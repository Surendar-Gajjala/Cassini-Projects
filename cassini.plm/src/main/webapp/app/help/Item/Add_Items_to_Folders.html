What is Adding item to folder?

            Adding an item to the folder is one of the best features of this application, this is nothing but organising related or needed items into a folder with a structure, a folder will may contain multiple objects like items, ECOs, etc… this is to manage different objects in a folder structure which helps user to get details easier and faster.

 

Why do we need to add items to folder?

            This action is not mandatory, but it helps user related or needed items into a folder structure and find all those objects easier and faster.

 

How to add items to folder?

 

1. To add items into folder, select Items

2. Click on "Add To Folder"

3. Select folder, (Items added to Folder successfully).
