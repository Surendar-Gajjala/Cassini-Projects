What is Exporting items?

            Items exporting is one of the great features of this application, user can select one or multiple items and export those into excel CSV format, this feature will help users to download required items.

Why do we need to Export items?

            Export items is not a mandatory operation, it helps user to download, store

How to Export items?

 

1. To export Items,Select items

2. Click on "Export"

3. Select "CSV", (Save to local storage)
