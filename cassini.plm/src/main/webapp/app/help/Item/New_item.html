What is ITEM creation?

            Typical manufacturing organisation will have 100s of items associated with its manufacturing process, whenever a new item comes into that process it needs to be created in the system for tracking. An item has attributes, BOM, where used, changes, files, manufacturer parts, related items, projects and requirements on it, all these details will help users to identify required information about a specific item.

Why do we need to CREATE Item?

            It is very important, and users can’t make any action without having an item. Here we have a dependency for item creation, first we should specify item types in classification and then only we can create items.

How to CREATE Items?

 

1.Click on "Navigation" icon.

2.Click on "Items",It will take user to the Items window.

3.Click on "New Item", Popup will appear.

4.Enter all the mandatory fields (Marked as *).

5. Click on "Create",Success message will appear.
