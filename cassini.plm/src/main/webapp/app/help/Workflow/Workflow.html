What is Workflow?

            Workflow is a series of works arranged in a flow. A workflow can be attached to ECO, at the time of change process, workflow describes the state of the change in defined change process.

 

Why do we need to create Workflow?

            Workflow is useful in identifying the state of a work, user can notify about their work in series of works. With help of workflow manufacturers can run their operations smooth and hassle free.

 

How to Create Workflow?

1. Select Workflow from navigation bar

 

2. Click on New Workflow

3. Start and Finish points will be displayed on screen

4. Click on Start point

5. Use displayed Append Task and Append Intermediate Task and draw a flow

6. Use Arrows to link one work to the other one

7. Select Workflow Type

8. Enter details Name, Description and Assign

 

Note: Any one of the work or tasks should be in Released state

9. Select created task

10. Select Status Type as Released

 

How to Delete Workflow?

1. Select Delete Workflow from Actions

2. Confirmation popup will open, click on ‘OK’ (Workflow will be deleted successfully)
