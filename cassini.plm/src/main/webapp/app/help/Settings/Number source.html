What is the use of Settings?

            Here users can create number sources, lifecycles, value lists, attributes, relationships and preferences. Settings tab is all about to setup application, with this information users can perform different operations of the application.

 

How to create number source?

 

1. Select Settings tab from navigation bar

2. Select number source

3. Click on “+” to create new number source

4. Enter details (name, description, numbers, Start, increment, pad with, prefix, suffix)

5. User can look at example for reference

6. Click on right mark (New number source will be created successfully)
