What is Change?

            Changes plays a major role in manufacturing, change can be any type, change types can be created in classification. Here user can create ECO and attach a workflow to it, based on this workflow team members will operate their tasks and notify on other changes.

 

How to create Change?

1. Select Changes from navigation

2. Click on New ECO

3. New ECO panel opens

4. Select Change type

4. Click on Auto to set a Change number

5. Select Owner

6. Enter Title and Description

7. Select Workflow

8. Click on Create (ECO will be created successfully)

 

How to View Change details?

1. Click on an ECO

 

2. User can see multiple tabs here (Basic details, Attributes, Workflow, Affected Items, and Files)

 

3.  Click on Attributes to view Attributes & its values on ECO (User can enter Attributes values here)

 

4. Click on Workflow to view ECO workflow and workflow status

 

5. Click on Affected Items to view items that are affecting from this change

 

6. Click on Files to view uploaded files to ECO

 

 

How to Promote/Demote ECO?

1. Click on an ECO

 

2. Select Workflow tab

3. Click on Start Workflow

 

4. User can identify current status of the ECO (Highlighted as Yellow)

5. Click on Promote/Demote to process the flow (Finished works are highlighted as Green)

 

How to Delete ECO?

1. Select Delete ECO from Actions

 

2. Popup will appear for confirmation, click on ‘OK’ (ECO will be deleted successfully)
