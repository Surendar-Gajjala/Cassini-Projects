Home screen provides access to entire PLM functions and it has
  1.     Navigation Bar

  2.     Top search, it allows user to search different objects from home screen

  3.     Widget space

  4.     Add widgets

  5.     Login user info

 

User can customize HOME page view with different widgets.

What is Widgets?

Widget is a GUI (Graphical User Interface) which displays lot of information related to

·         Recently Visited pages

·         Logged in user’s tasks ‘My Tasks’

·         Latest revision updated items

·         Most used manufacturer parts

·         Released ECO’s and

·         Saved searches

How to Add Widgets?

1.    Go to Home page

2.    Click on “+” button

3.    Click on the Checkbox of your choice to be displayed on Home Screen

4.    Click on Add button

5.    Selected Widget will be added to Home Screen successfully with corresponding msg displayed
