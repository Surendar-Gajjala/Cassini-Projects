LOGIN

Lets start with Login to Cassini.TEST web application.

Steps to Login

  • Open the Browser - Chrome / Firefox

  • Type in the URL in the browser - https://test.cassiniqa.com

  • User will see the Login Screen
  1. Fill in the User ID
  2. Fill in the Password
  3. Click the Login Button
User will be successfully logged into the Cassini.TEST Web application
Note: If you are unable to login, please contact the Admin
