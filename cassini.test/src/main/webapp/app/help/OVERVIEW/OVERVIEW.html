OVERVIEW

Cassini.TEST is a fully configurable automation testing solution from Cassini Systems for product manufacturing organizations. Cassini TEST is an end to end solution for managing the Product Testing, Test Scenario, Test Plan, Test Suite, and Test Cases. Cassini TEST is designed to test a product configuration through test scenarios and with test cases, it will help manufacturing organization to automate its testing process. Organizations can define/write execution file codes within the TEST solution, it can be a program or script (Groovy and Python).

Cassini Systems is an India based IT Solutions provider supporting multiple industry verti-cals - Manufacturing, Infrastructure - Railways, Logistics and many more.
