Test Case Execute with Program

Steps to execute test case with program

  1. Select execution type (Program)
  2. Enter Program Name
  3. Enter Params
  4. Copy and Paste work directory
  5. Click on Save button

  6. Create Input Param types
  7. Click on button
  8. Type in Name and Description
  9. Select Data Type
  10. Click on symbol to save
  11. Create output Param types
  12. Click on button
  13. Type in Name and Description
  14. Select Data Type
  15. Click on symbol to save
