Test Case Execute with Script

Steps to execute test case with script

  1. Select script
  2. Select language (Groovy or Python)
  3. Click to go groovy script editor
  4. Enter code
  5. Click on create
  6. Click on Save button
  7. Create Input Param types
  8. Click on button
  9. Type in Name and Description
  10. Select Data Type
  11. Click on symbol to save
  12. Create output Param types
  13. Click on button
  14. Type in Name and Description
  15. Select Data Type
  16. Click on symbol to save
