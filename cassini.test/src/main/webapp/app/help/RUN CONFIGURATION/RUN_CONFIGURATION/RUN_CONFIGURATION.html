Run Configuration

Steps to run configuration

  1. Click on Test Definition
  2. Right click on Scenario
  3. Select Export (it will generate TestCasesExport Excel file)
  4. Save exported sheet in local and
  5. Add columns to sheet [Run Configuration (must be a Run Configuration name given in application), Result and Date]
  6. Add system properties to sheet if you want
  7. Enter values to added columns
  8. Save Excel sheet
