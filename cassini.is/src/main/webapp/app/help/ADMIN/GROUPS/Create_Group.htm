NEW GROUP

 

1. Go to Admin module.

2. Right click on “Groups”, select New Group.

3. Enter “Name”, “Description”.

4. Click on Select User then it will pop up a window.

5. Check the users whom you are going to assign that role.

6. Click on “ADD”.

7. Can see selected users under “Selected user” List.

8. Click on Create, will get “Group created successfully “message.
