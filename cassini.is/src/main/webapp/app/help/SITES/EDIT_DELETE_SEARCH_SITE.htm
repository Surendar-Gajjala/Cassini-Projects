EDIT/DELETE/SEARCH SITE

 

1.Click on Edit button,Description field enable to edit,if required edit description click on save.

2.To delete site,click on "Delete",Click on OK.

3.To search Site enter site into search text box.

4.Click on "Search".

5.To clear search Click on Search clear button.
