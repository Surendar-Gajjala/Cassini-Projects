STEPS TO ADD MATERIALS TO THE TASK

 

1.Click on task name.

2.Click on "Material".

3.Click on "Add Material", popup will appear.

 

 

4.Select Material.

5.Click on "Add" will get successful message.

6.Click on Cancel to close the current window.

 

7.Enter "Task Qty".

8.Click on "Delete" button to delete added material.

9."Task Qty" Should not be grater than "Balanced Qty".

 

 

 

9. Repeat the same steps to add Machine, Manpower.

 

Add files into the Task:

 

1.Click on "Files".

2.Click on "Add Files".

3.Click on "Add Files",Select files from the local machine.

4."Go Back" helps to back to the previous menu.

 

 

5.Click on "Start upload" will get success message.

6.To cancel the upload Click on "Cancel Upload".

7.To cancel the selected file Click on "Cancel" .
