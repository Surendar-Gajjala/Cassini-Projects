UPDATE STATUS OF THE TASK

1. Select a Task of your choice,then it's taken to a new window.

 

 

1.Task detail view contains,

A. Basic.

B. Manpower.

C. Material.

D. Machine.

E. Files

A. Basic :

1. Click on “Basic”.

2. Click on task will start and then start button  became  Finish .

3. Status will change from "ASSIGNED"  to "IN PROGRESS".

4. User can edit 'Name', 'Description', 'Assigned to' ,"Unit of Work","Total Unit of work" by click on their names text box will appear enter the require name then click on   

     'Submit',else click on 'cancel ' to go back to the previous status.

5.Click on Add an entry(+) .

6.Enter "Units Completed"(mandatory).

7.Enter "Notes"(optional).

8.Click on "Save".

9.Click on "Cancel" to cancel the entry.

 

 

 

9. "Total Unit of Work" should equals to "Units Completed",if both are equal Task changes to Finished Status,Until Status should be In progress.

10.Units completed Updated depends upon the "Units Completed"

11.Click on 'Back' to go back to the previous window.
