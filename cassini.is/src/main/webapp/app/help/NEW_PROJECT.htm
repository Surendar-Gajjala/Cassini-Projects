NEW PROJECT

1.Click on "New Project",.popup will appear.

2.Enter "Name".

3.Enter "Description (optional)".

4.Click on "Portfolio" drop down.

    a. Select Portfolio from list.

5.Click on "Project Owner" Drop down.

    b. Select Project Owner from list.

6.Enter "Planned Start Date".

7.Enter "Planned Finish Date".

8.Click on "Create".

9.Will get "Project created Successfully" message.
