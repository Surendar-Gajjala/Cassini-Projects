CREATE LIST OF VALUES

 

1.Click on "Navigation".

2.Click on "Settings",Its taken to the new window.

3.Click on List of Values.

4.Click on New Lov (+) button.

5.Update the Lov name.

6.Click on OK button.

7.Click on "Cancel" to close the lov.

 

We can add a value to the LOV as we wish.

 

STEPS TO ADD VALUES TO LOV

 

1.    Click on Add value button

2.    Enter the Lov name

3.    Click on OK button

 

Similarly, we can add more values to LOV.
