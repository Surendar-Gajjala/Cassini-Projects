CREATE STORE

 

1.Click on "Navigation" icon.

2.Click on "Stores" under Procurement.

 

 

CREATE/DELETE/SEARCH STORE

 

3.Click on "New Store",Popup will appear.

4.Enter Details.

5.Click on "Create",Will get Successful message.

6.Click on "X" cancel to quit from the current window.

 

 

DELETE /SEARCH STORE:

 

1.To delete Store Click on "Delete" button.

2.To search Store enter any store details into Search text box click on "Search".

3.To clear the Search mode click on "Clear Search (X)" button.

4.By click on "Store" name it's taken o the Store detail view window.
