ISSUES

 

1.Click on "Issues" functionality.

2.Click on "New Issue",Popup will appear.

3.Click on "Auto" number will appear.

4.Select Project.

5.Select "WBS".

6.Select "Task".

7.Select "Assigned to".

8.select "Issue Date".

9.Enter "Notes"(optional).

10.Click on "Create",Will get Success message.

 

 

STEPS TO PRINT ISSUE CHALLAN:

 

1.Click on Issue "Number".

 

 

2.Click on "Items".

 

 

3.Click on "Print Issue Challan".

 

 

4.Click on "Save" ,file will start to download.

5.open downloaded file click on Print.
