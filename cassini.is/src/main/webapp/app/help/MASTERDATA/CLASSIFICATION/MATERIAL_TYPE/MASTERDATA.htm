STEPS TO CREATE MATERIAL TYPE

 

1.Click on "Navigation" icon.

2.Ckic on "Master Data".

 

 

3.Right Click on “Material”.

4.click on  “Add Type”.

5.Enter “Name” (Mandatory).

6.Enter “Description”(optional).

7.Select "Auto Number Source" from list.

8.Click on “Save”, Will get “Material Type Saved Successfully” message.
