STEPS TO CREATE NEW MANPOWER

 

1.Click on Navigation.

2.Click on "Master Data".

 

 

3.Click on “MANPOWER”.

4.Click on “New Manpower”.

5.Select "Manpower Type".

6.Click on "Auto" to generate "Manpower Number".

7.Check "Existing" button.

8.Select "Person".

9.Enter Descriptio (optional).

10.Click on "Create".

11.Click on ‘X’ to cancel the current process.
