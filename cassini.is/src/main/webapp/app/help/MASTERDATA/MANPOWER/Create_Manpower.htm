STEPS TO CREATE NEW MANPOWER

 

1. Click on Navigation.

2. Click on "Master Data".

 

 

3.Click on “MANPOWER”.

4.Click on “New Manpower”.

5.Select "Manpower Type".

6.Click on "Auto" to generate "Manpower Number".

 

CREATE MANPOWER WITH EXISTING USER

 

7.Check "Existing" button.

8.Select "Person".

9.Enter Descriptio (optional).

10.Click on "Create".

11.Click on ‘X’ to cancel the current process.

 

 

 

CREATE MANPOWER WITH NEW USER

 

Repeat Steps 1,2,3,4,5,6.

7.Check "New" button.

8.Enter "First Name"(Mandatory).

9.Enter "Last Name"(Optional).

10.Enter Phone (Option).

11.Enter "E-mail"(optional).

12.Enter "Description (optional)".

13.Click on "Create".

14.Click on ‘X’ to cancel the current process.
